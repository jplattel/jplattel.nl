Fundsup

31-05-2018
python django , api , vue , postgres

To enable target marketing towards startups and investors alike, I helped Fundsup create a single database for campaigns and meta-data for their platform. The main technology of this stack was python & Django for the backend and the API and Vue for the front-end. The database was PostGres with JSON fields allowing for querying non-schema based fields.
